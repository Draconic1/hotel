# Сервис для онлайн-бронирования отелей

  Данный сервис предлагает возможность бронировать различные номера в отелях любого класса под ваш комфорт. Бронь отелей доступна только зарегистрированным пользователям, гость может только просмотреть предлагаемые услуги. 
    
  Стек технологий:
  Frontend: React, Redux-tookit, axios
  Backend: Node.js + Express  
  Database: SQLite

Интрфейс поддерживает разные возможности в соответствии с ролью пользователя.

**Гость:** - имеет стартовый функционал, просмотр основных страниц и возможность зарегистрироваться. 
**Авторизованный пользователь:** - имеет расширенный функционал с возможностью бронировать номера и просматривать собственные брони.  
**Администратор:** - имеет панель администратора для управления отелями и номерами, с возможностью добавлять и редактировать описания, просматривать брони пользователей, т.д.  
  В проекте использовалась база данных sqlite.  
  
### Запуск приложения    
1. ```console
   cd frontend   
   npm start
   ```

       
2. ```console
   cd backend    
   npm start
   ```
